Convert JCL Parms to CoSort SortCL Scripts

Demonstrate these conversions by performing the following steps:

  1. From the CoSort menu, open the "Import JCL Sorts" wizard.
  2. In the From directory field, browse to the JCL_SORT_Convert\data folder in the workspace for the source.
  3. Select mvs1 and mvs2 as the sort cards to convert.
  4. In the Into folder field, define the target, using Browse, as the JCL_SORT_Convert\script folder as the target directory.
  5. In the Source Format field, select MVS.
  6. Note the Options check boxes available.
  7. Click Finish and examine the target .scl files.
